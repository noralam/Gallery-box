All gallery items will be open nice and smooth lightbox. </p>

#### How to use gallery Box

#### Add image gallery item:-
<ol style="margin-top:20px">
								<li><strong>Enter image gallery name -</strong>Type your image gallery name.</li>
								<li><strong>Click to add image gallery item - </strong>Click this button and add your image gallery items.</li>
								<li><strong>Enter gallery item title -</strong> Type image gallery item title here.</li>
								<li><strong>Set gallery image - </strong> This image show in front.Image size (300*300) for 4 column,(450*450) for 3 column, (600*600) for 2 column.All images use same size for better view.</li>
								<li><strong>Set lightbox Image - </strong>Set lightbox Image This image show when lightbox open.If you don't set this image, gallery image will be open lightbox.</li>
								<li><strong>Enter image lightbox caption - </strong>Lightbox caption should be simple.Default caption is item title.You can hide Caption by gallery box settings </li>
								<li><strong>Enter button text -</strong>Button text must be small.Default button text is( view large ) </li>
								<li><strong>Add more -</strong>Click add more button and add more gallery image item.</li>
								<li><strong>Publish -</strong>Finally publish this item.</li>
								<li><strong>Shortcode -</strong>After publish this gallery you will see new shortcode for this gallery show in front.</li>
							</ol>
#### Add Youtube video gallery item:-
<ol style="margin-top:20px">
								<li><strong>Enter gallery item title -</strong>Type your youtube gallery item title here.</li>
								<li><strong>Enter video ID -</strong> Go Youtube and select your video than copy Youtube video id from url. Now past id this box.</li>
								<li><strong>Set Youtube Gallery Image -</strong> Image size (300*300) for 4 column,(450*450) for 3 column, (600*600) for 2 column.All images use same size for better view..If you want to show Youtube default thumbnail keep empty this box.</li>
								<li><strong>Enter video lightbox caption -</strong>lightbox caption should be simple.Default caption is item title.You can hide Caption by gallery box settings </li>
								<li><strong>Enter youtube video width-</strong>Youtube video lightbox width. Default width 600 ).</li>
								<li><strong>Enter youtube video height-</strong>Set youtube video. Default height 400 ).</li>
								<li><strong>Enter button text -</strong>Button text must be small.Default button text is( view large ) </li>
								<li><strong>Enter button text-</strong>Button text must be small.Default button text is( Show video )</li>
								<li><strong>Click add more button-</strong>Click add more button for add new Youtube video gallery item. </li>
							</ol>

<ol style="margin-top:20px">
								<li><strong>Click to add vimeo video gallery item -</strong>Click this button and vimeo video for create gallery.</li>
								<li><strong>Enter vimeo video title -</strong> Type your vimeo video title here.</li>

								<li><strong>Set vimeo gallery image -</strong> Image size (300*300) for 4 column,(450*450) for 3 column, (600*600) for 2 column.All images use same size for better view..If you want to show Vimeo default thumbnail keep empty this box.</li>
								
								<li><strong>Enter video lightbox caption -</strong>lightbox caption should be simple.Default caption is item title.You can hide Caption by gallery box settings </li>
								<li><strong>
								Enter Vimeo video width-</strong>Vimeo video lightbox width. Default width 600 ).</li>
								<li><strong>
								Enter Vimeo video height-</strong>Set Vimeo video. Default height 400 ).</li>
								
								<li><strong>Enter button text-</strong>Button text must be small.Default button text is( Show video )</li>

#### Add Soundcloud audio gallery item:-
<ol>
								<li><strong>Enter gallery item title -</strong> Type your Soundcloud item title here.</li>
								<li><strong>Enter soundcloud audio id -</strong> Go soundcloud website and select your audio than copy soundcloud audio id from share embed code. Now past id this box.</li>
								<li><strong>Set Soundcloud Gallery Image -</strong> Image size (300*300) for 4 column,(450*450) for 3 column, (600*600) for 2 column.All images use same size for better view.</li>
								<li><strong>Enter Soundtrack lightbox caption -</strong>lightbox caption should be simple.Default caption is item title.You can hide Caption by gallery box settings. </li>
								
								<li><strong>Enter button text-</strong>Button text must be small.Default button text is(Listen song )</li>
								<li><strong>Add more -</strong> Click add more and add more sound for Soundcloud gallery.</li>
							</ol>
#### Add iframe gallery item:-
<ol style="margin-top:20px">
								<li><strong>Click to add iframe gallery item -</strong> Click and add iframe gallery item for create iframe gallery.</li>
								<li><strong>Enter iframe gallery item title -</strong> Type your iframe gallery item title here.</li>
								<li><strong>Set iframe Gallery Image -</strong> Image size (300*300) for 4 column,(450*450) for 3 column, (600*600) for 2 column.All images use same size for better view.</li>
								<li><strong>Enter page url -</strong> Go your web page and copy page link then past page link in url box.</li>
								<li><strong>Enter iframe lightbox caption -</strong>lightbox caption should be simple.Default caption is item title.You can hide Caption by gallery box settings. </li>
								<li><strong>Enter iframe width -</strong>Set your iframe width here.default width 600px </li>
								<li><strong>Enter iframe height -</strong>Set your iframe height here.default height 400px </li>
								<li><strong>Enter button text-</strong>Button text must be small.Default button text is(SHOW IFRAME)</li>
								<li><strong>Add more-</strong> Click add more button and add more items for create iframe gallery.</li>
							</ol>

== Installation ==

<h2>This section describes how to install the plugin and get it working.</h2>

e.g.

<h4>First things first, thank you for choosing our Gallery Box lite plugin!</h4>
<h4>You can install the plugin one of two ways:</h4>
<h4>Install using FTP</h4>
<ol>
	<li>Unzip the Gallery Box.zip file locally to your machine</li>
	<li>Connect to your FTP area using your preferred FTP package</li>
	<li>Upload the Gallery Box folder that you extracted from the zip file to the plugin folder of your WordPress installation (wp-content -> plugin)</li>
	<li>Go to plugin and activate Gallery Box.</li>
</ol>
<h4>Install by WordPress (recommended)</h4>
<ol>
	<li>Login to your WordPress admin area.</li>
	<li>Navigate to plugin and click the Add New button at the top</li>
	<li>Click the Upload plugin button at the top</li>
	<li>Click the Choose File button and locate the Gallery Box.zip </li>
	<li>File on your machine and click the Install Now button </li>
	<li>On the basis the plugin installs correctly, click the Activate link.</li>
</ol>
